# High-Performance Order Matching Engine

A low-latency electronic order matching engine built in C17.

## Features

- Buy Order Book
- Sell Order Book
- Price-Time Priority
- Trade Execution
- Partial Fills
- Order Cancellation
- CSV Trade Logging
- Performance Benchmarking
- FastAPI Integration
- React Dashboard

## Tech Stack

- C17
- GCC
- Git
- FastAPI
- React
- Vercel
- Render
